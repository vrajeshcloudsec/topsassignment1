What is an Input Device?

An input device is a hardware device used to enter data into a computer.

**Examples:**

* Keyboard
* Mouse
* Scanner
* Microphone
* Webcam

---

## 2. What is an Output Device?

An output device shows the result from the computer.

**Examples:**

* Monitor
* Printer
* Speaker
* Projector

---

## 3. What is CPU?

CPU (Central Processing Unit) is called the “brain of the computer.”
It processes instructions and controls all operations.

---

## 4. What are the Types of CPU?

* Single Core CPU
* Dual Core CPU
* Quad Core CPU
* Hexa Core CPU
* Octa Core CPU

---

## 5. What Do We Need to Keep the CPU Healthy?

* Clean dust regularly
* Use thermal paste
* Keep cooling fan working
* Avoid overheating
* Use proper power supply

---

## 6. Practical: Remove Processor and Apply Thermal Paste

1. Shut down computer and unplug power.
2. Open cabinet.
3. Remove CPU cooler/fan.
4. Unlock processor socket and remove CPU carefully.
5. Clean old thermal paste.
6. Apply small amount of new thermal paste.
7. Reinstall CPU correctly.
8. Lock socket and install cooler again.
9. Start the system.

---

## 7. Practical: Identify CPU and CPU Socket

1. Open computer cabinet.
2. Locate processor on motherboard.
3. Check socket type written near CPU or in BIOS/software.
4. Examples of sockets: LGA1151, AM4.

---

## 8. What is Memory?

Memory stores data and instructions for the computer.

---

## 9. What are the Types of Memory?

### Primary Memory

* RAM
* ROM

### Secondary Memory

* Hard Disk
* SSD
* Pen Drive

---

## 10. Practical: Identify Memory Types

1. Open cabinet.
2. Locate RAM slots.
3. Check RAM label (DDR3, DDR4, DDR5).
4. Identify storage devices like HDD or SSD.

---

## 11. Practical: Install Memory in System

1. Turn off computer.
2. Open RAM slot clips.
3. Align RAM notch correctly.
4. Press RAM gently until locked.
5. Start computer and check detection.

---

## 12. Practical: Identify Main Memory Frequencies

1. Open BIOS or use software like CPU-Z.
2. Check RAM frequency.
3. Example: 2400 MHz, 3200 MHz.

---

## 13. What is BIOS?

BIOS (Basic Input Output System) is firmware stored on motherboard.
It starts the computer and checks hardware.

---

## 14. Working Process of BIOS

1. Power ON computer.
2. BIOS performs POST (Power On Self Test).
3. Detects hardware devices.
4. Loads bootloader from storage device.
5. Starts operating system.

---

## 15. Practical: Reset BIOS When System is ON

1. Restart computer.
2. Enter BIOS setup using Del/F2 key.
3. Select “Load Default Settings.”
4. Save and exit.

---

## 16. Practical: Hard Reset BIOS

1. Turn off computer and unplug power.
2. Remove CMOS battery for 5 minutes.
3. Reinsert battery.
4. Start computer.

---

## 17. Practical: Identify BIOS Chip

1. Open motherboard cabinet.
2. Locate small chip labeled BIOS/UEFI.
3. Usually near CMOS battery.

---

## 18. What is CMOS?

CMOS stores BIOS settings like date, time, and boot settings.

---

## 19. What is Motherboard?

Motherboard is the main circuit board connecting all computer parts.

---

## 20. Types of Motherboard

* ATX
* Micro-ATX
* Mini-ITX

---

## 21. Practical: Identify Parts of Motherboard

Identify:

* CPU socket
* RAM slots
* PCI slots
* SATA ports
* CMOS battery
* BIOS chip

---

## 22. Practical: Remove Removable Parts from Motherboard

1. Power off system.
2. Remove RAM.
3. Remove CPU fan.
4. Remove storage cables.
5. Remove expansion cards carefully.

---

## 23. What is System Bus?

System bus transfers data between CPU, memory, and devices.

---

## 24. What is Chipset and Types of Chipset?

Chipset controls communication between components.

### Types:

* Northbridge
* Southbridge

---

## 25. Working of Northbridge Chipset

Northbridge connects CPU with:

* RAM
* Graphics card
* High-speed devices

---

## 26. What is SMPS and Its Purpose?

SMPS (Switched Mode Power Supply) converts AC power into DC power for computer parts.

---

## 27. Practical: Install SMPS

1. Open cabinet.
2. Place SMPS in cabinet slot.
3. Tighten screws.
4. Connect motherboard and device power cables.

---

## 28. How to Check SMPS?

* Check fan rotation
* Use SMPS tester or multimeter
* Check if system powers ON

---

Storage devices save data permanently and provide data when needed.

## 31. Practical: Remove and Reinstall Storage Device and Make GPT Disk

## 32. What is SATA?

SATA (Serial ATA) is an interface used to connect storage devices.

---

## 33. Working of SATA

SATA transfers data between motherboard and storage devices.

---

## 34. Practical: Install SATA

1. Connect SATA data cable to motherboard.
2. Connect SATA power cable from SMPS.
3. Start computer and check detection.

---

## 38. What is Boot Process?

Boot process is the process of starting the computer and loading the operating system.

---

## 39. Boot Process in Linux

1. BIOS/UEFI starts system
2. GRUB bootloader loads
3. Linux kernel loads
4. System services start
5. Login screen appears

---
